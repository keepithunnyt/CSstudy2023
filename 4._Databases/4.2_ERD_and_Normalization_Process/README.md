# 4.2.1 ERD의 중요성
- ERD : Entity Relationship Diagram. 릴레이션간의 관계 정의.
- ERD를 기반으로 데이터베이스를 구축하며, 디버깅이나 프로세스 재설계가 필요한 경우에 설계도 역할을 한다.
- 단점 : 비정형 데이터를 충분히 표현할 수 없다.
  - 비정형 데이터 : 비 구조화 데이터를 말하며, 미리 정의된 데이터 모델이 없거나, 정의된 방식으로 정지되지 않은 정보를 의미.
